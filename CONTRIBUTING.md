
# Contributing to RansomWatch

Thank you for your interest in contributing!

## Development Setup

1. Fork the repo and clone it to your local machine.
2. Follow the "Quick Start (Local)" instructions in README.md.
3. Create a new branch for your changes:
   ```bash
   git checkout -b your-feature-branch
   ```
4. Make your changes and commit them with descriptive commit messages.
5. Push to your branch and open a pull request!

## Code Style

- **Backend/Frontend**: ESLint + Prettier (make sure to run `npm run lint`)
- **ML Service**: Black + Flake8 (optional but recommended)

## Testing

- Run backend tests: `cd backend && npm test`
- Run frontend build: `cd frontend && npm run build`

## Pull Requests

1. Ensure your changes are tested.
2. Update the README if needed.
3. Describe your changes in detail in the PR description.
4. Link to any related issues.
